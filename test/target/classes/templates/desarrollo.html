API Services Documentation

1. Crear una cuenta (Create)

Descripción: Permite crear una nueva cuenta de usuario con los detalles proporcionados.

Método: POST

URL: http://localhost/crear-cuenta

Parámetros: Ninguno

Body:

{
                "nombre": "nombre",
                "apellido": "apellido",
                "correo": "b@gmail.com",
                "contraseña": "123",
                "arrendador": true
            }

Respuesta esperada: Código 200 en caso de éxito. Detalles del nuevo usuario registrado.

2. Registrar una propiedad (Create)

Descripción: Permite registrar una nueva propiedad de un arrendador.

Método: POST

Parámetros: Id: ID del arrendador

URL:http://localhost/arrendador/{id}/registrar-propiedad

Body:

{
                "nombrePropiedad": "casa del shitpost",
                "departamento": "A",
                "municipio": "Medellín",
                "tipoIngreso": "Alquiler",
                "descripcion": "Hermosa casa con vista al lago y amplios espacios.",
                "cantidadHabitaciones": 3,
                "cantidadBaños": 2,
                "permite_mascotas": true,
                "tiene_piscina": false,
                "tiene_asador": true,
                "valor_noche": 50000
            }

Respuesta esperada: Código 200 en caso de éxito. Detalles de la propiedad registrada.

3. Calificar Arrendador (Update)

Descripción: Permite calificar a un arrendador utilizando el ID del arrendador y la calificación.

Método: POST

URL:http://localhost/calificaciones/arrendador/1?calificacion=2

Parámetros: ID del arrendador, Calificación asignada al arrendador

Respuesta esperada: Código 200 en caso de éxito. Confirmación de que la calificación se ha asignado correctamente.

4. Modificar Propiedad (Update)

Descripción: Permite modificar una propiedad registrada por un arrendador.

Método: PUT

URL:http://localhost/arrendador/1/modificar-propiedad/1

Body:

{
                "nombrePropiedad": "Row Row Fight the power",
                "departamento": "que carajo es un kilómetro?",
                "municipio": "natlan",
                "tipoIngreso": "sí",
                "descripcion": "Hermosa casa con vista al lago y amplios espacios.",
                "cantidadHabitaciones": 3,
                "cantidadBaños": 2,
                "permite_mascotas": true,
                "tiene_piscina": false,
                "tiene_asador": false,
                "valor_noche": 50000
            }

Respuesta esperada: Código 200 en caso de éxito. Detalles de la propiedad actualizados.

5. Iniciar Sesión (Retrieve)

Descripción: Permite iniciar sesión con el correo y contraseña proporcionados.

Método: GET

URL:http://localhost/iniciar-sesion/{email}/{contraseña}

Parámetros: Correo: Correo electrónico del usuario, Contraseña: Contraseña del usuario

Respuesta esperada: Código 200 en caso de éxito. Redirección a la página del arrendador o arrendatario según el tipo de usuario.

6. Obtener las propiedades de un Arrendador (Retrieve)

Descripción: Devuelve una lista de todas las propiedades registradas por un arrendador.

Método: GET

URL:http://localhost/arrendador/1/propiedades

Parámetros: Id: ID del arrendador

Respuesta esperada: Código 200 en caso de éxito. Lista de propiedades del arrendador.
